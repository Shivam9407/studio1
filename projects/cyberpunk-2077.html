01 / The Brief

Overview

For Cyberpunk 2077's Phantom Liberty expansion, CD Projekt RED commissioned us to redesign the entire in-game HUD and diegetic UI. Every screen, every holographic display, every neural-link interface needed to feel like it belonged to a world 57 years from now — functional, gritty, and alive.

02 / Context

The
Challenge

The original UI had been criticized for feeling too sterile. The challenge was to rebuild from scratch — creating a system that felt lived-in, glitched, and culturally rich. Every element needed to reflect the fictional mega-corporations, street gangs, and neural implants of Night City.

Performance was non-negotiable: the UI had to run flawlessly inside a GPU-intensive open-world game engine, using only compositor-optimized layers with zero JavaScript overhead.

03 / Execution

The Solution

We built 4 distinct UI language families — one for each mega-corp — each with its own glyph system, animation cadence, and glitch signature.

01

Glitch Language

A procedural glitch system in HLSL — every UI element corrupts based on in-world hacking events and cyberware status.

02

Corp Theming

Militech, Arasaka, Biotechnica — each faction's UI uses custom iconography and animation timing based on their cultural identity.

03

Diegetic Integration

HUD elements exist inside the world. Holographic menus cast light on surfaces, neural link messages appear embedded in the environment.
