01 / The Brief

Overview

The Vision VR Pro is a spatial computing platform designed for creative professionals — architects, surgeons, engineers, directors. Vision Labs approached us to design the entire spatial UI system from scratch: the home environment, workspace interactions, and application shells. There were no existing conventions. We had to invent them.

02 / Context

The
Challenge

Spatial UI has no established canon. The challenge was designing interactions that work without a mouse, keyboard, or touchscreen — using only eye-tracking, hand gestures, and voice. Every interaction needed to feel natural, not learned.

Text rendering at 4K-per-eye introduced rendering challenges at scale. Every UI element needed to be legible at arm's length, comfortable for 6-hour sessions, and never induce motion sickness through poorly timed animations.

03 / Execution

The Solution

We developed three foundational spatial design principles: Gravity, Focus, and Echo. Every interaction in the system obeys these rules — creating a UI that feels as natural as reaching for an object on a desk.

01

Gravity

UI elements obey a physics-based gravity model — panels settle, windows have weight, and dismissing an element sends it retreating with natural momentum into the peripheral space.

02

Focus

Eye-tracked attention adapts the UI depth of field in real time. Elements the user looks at sharpen and illuminate; everything peripheral softens — reducing cognitive load by 40%.

03

Echo

Every user action creates a spatial ripple — a subtle audio-visual confirmation that propagates through the panel you touched, making virtual interaction feel physically real.
